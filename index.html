Getting Started

1

Download FreePay POS

Get the latest version from our GitHub repository.

2

Flash to microSD

Write the image to a microSD card using your preferred tool.

3

Boot & Configure

Insert the card and power on. The system boots directly into the app.

4

Start Accepting Payments

Connect compatible wallets and begin processing transactions.
